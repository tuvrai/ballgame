# ballgame
simple arcanoid like game in c++

to use put files in this hierarchy:
## gamedata
  levels.txt
  ### img
    ball.png
  ### fonts
    JosefinSans-Regular.ttf
  ### levels
    level1.txt
    level2.txt
    level3.txt
    level4.txt



instr:
### h 
show/hide HUD
### p
pause game
### up arrow
change focus (velocity will change more in x or y dir)
### left/right
move racket
